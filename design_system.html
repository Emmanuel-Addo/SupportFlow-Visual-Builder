SupportFlow AI - UI Design System

Visual language specifications for the decision-tree decision builder editor canvas, cards, connectors, and semantic layout rules.

FRAME 01

The Canvas

The main editing environment consists of an infinite scrolling board mapped with absolute layout coordinates. To avoid layout clutter, we use a solid background system that emphasizes node interactions.
